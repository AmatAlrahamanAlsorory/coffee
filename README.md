# نظام كاشير ومبيعات المقهى الذكي ☕

نظام متكامل لإدارة مبيعات المقهى باستخدام Streamlit.

## المميزات الرئيسية

### 📊 لوحة التحكم
- عرض المؤشرات المالية (إجمالي المبيعات، عدد الأكواب، متوسط التقييم)
- رسوم بيانية تفاعلية (أعمدة ودائرية)
- فلاتر متقدمة (حسب المنتج والفترة الزمنية)
- تصدير البيانات (Excel و CSV)

### ☕ المنيو والبيع
- عرض قائمة المنتجات مع صور
- نظام فاتورة تفاعلي
- إضافة المنتجات للطلب بضغطة زر
- ترحيل المبيعات تلقائياً

### ➕ إدارة المنتجات
- إضافة منتجات جديدة يدوياً
- التحقق من صحة المدخلات
- معالجة تكرار المنتجات
- تحديث المنتجات الموجودة

### ⚙️ الإعدادات والأمان
- نظام مصادقة بسيط
- نسخ احتياطي تلقائي
- إدارة ذاكرة التخزين المؤقت
- معلومات النظام

## التثبيت

1. تثبيت المتطلبات:
```bash
pip install -r requirements.txt
```

2. تشغيل التطبيق:
```bash
streamlit run app.py
```
